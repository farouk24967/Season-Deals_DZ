# Season Deals DZ - Réplique E-commerce

Ce projet est une réplique fonctionnelle d'un site de e-commerce type "Season Deals DZ", construite uniquement avec HTML, CSS (Tailwind) et JavaScript.

## Fonctionnalités

*   **Design Responsive** : S'adapte parfaitement aux mobiles, tablettes et ordinateurs.
*   **Catalogue Produits** : Affichage dynamique des produits par catégories.
*   **Panier Dynamique** : Ajout/Suppression de produits sans rechargement de page.
*   **Calcul Automatique** : Total du panier mis à jour en temps réel.
*   **Simulation de Commande** : Formulaire de checkout complet (Nom, Téléphone, Wilaya) simulant une commande "Paiement à la livraison".
*   **Animations** : Transitions fluides et animations d'entrée.

## Structure du Projet

*   `index.html` : La structure principale de la page.
*   `js/main.js` : Contient toute la logique (données produits, gestion du panier, événements).
*   `css/style.css` : Styles personnalisés et animations.

## Comment l'utiliser

1.  Ouvrez simplement le fichier `index.html` dans votre navigateur.
2.  Naviguez dans les produits, filtrez par catégorie.
3.  Ajoutez des articles au panier.
4.  Ouvrez le panier et cliquez sur "Passer la commande".
5.  Remplissez le formulaire pour tester la validation.

## Personnalisation

Pour changer les produits, ouvrez `js/main.js` et modifiez le tableau `products` au début du fichier. Vous pouvez changer les images, les prix et les descriptions facilement.
